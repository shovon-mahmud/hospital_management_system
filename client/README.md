# HMS Client (React + Vite)

## Scripts
- npm run dev — dev server on http://localhost:5173
- npm run build — production build
- npm run preview — preview build
- npm test — Vitest + RTL

## Env
Copy `.env.example` to `.env` and set `VITE_API_URL`.

## Features
- React Router v6
- Redux Toolkit
- Tailwind CSS
- Protected routes by role
- Toast notifications
